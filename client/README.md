# Obsidian Client

This is the frontend for Obsidian, built with React, Vite, and Shadcn UI.

## Quick Start

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Run development server:
   ```bash
   pnpm dev
   ```
3. Build for production:
   ```bash
   pnpm build
   ```

*This client requires the Obsidian Backend to be running.*
